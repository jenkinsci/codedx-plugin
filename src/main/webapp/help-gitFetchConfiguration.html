If Git has been configured for this project within Code Dx, enable to allow Code Dx to fetch the latest source from Git before analysis. This field is required if no input or tool output files have been specified.

Note: this will not cause the repository to be cloned in the Jenkins workspace - it will be cloned / fetched within Code Dx before analysis begins. File input fields such as "Source and Binary Files" and "Tool Output Files" will not be affected by the Git branch fetched within Code Dx.

Note: this field has no interactions the "Target branch" field, which specifies the project's Code Dx branch to associate findings with.
